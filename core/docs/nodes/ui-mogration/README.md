# Core-Rendered Node UI Migration

Status: Not developed

This folder documents what nodes must provide so Hexe Core can migrate from node-owned operational UIs to a Core-owned rendered UI.

The task list for implementation remains Core-only in `docs/New_tasks.txt`. Documents in this folder are handoff contracts for node authors and should not be read as instructions to change node repositories during the Core infrastructure phase.
